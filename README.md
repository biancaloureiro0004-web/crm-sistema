# Sistema CRM - Gestão Interna de Leads e RH 🚀

Este é um sistema de CRM desenvolvido para otimizar a gestão interna de uma empresa, permitindo o controlo de funcionários, leads, relatórios, recursos humanos e um calendário dinâmico.

## 🛠️ Tecnologias Utilizadas
- **Backend:** PHP
- **Base de Dados:** MySQL (Ficheiro .sql incluído na raiz para importação)
- **Frontend:** HTML5, CSS3, JavaScript / Bootstrap

## 📂 Funcionalidades
- Autenticação de utilizadores (`login.php` / `logout.php`)
- Painel principal resumo (`dashboard.php`)
- Módulo de Recursos Humanos e Funcionários (`rh.php` / `funcionarios.php`)
- Gestão e conversão de contactos (`leads.php`)
- Relatórios estatísticos e Calendário (`relatorios.php` / `calendario.php`)
